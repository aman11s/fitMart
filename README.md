```
cd fitMart
npm install
npm start
```

## **Key Features**

- Homepage
- Product listing page
- Filters by
  - Price
  - Category
  - Ratings
  - Price - High to Low
  - Price - Low to High
- Wishlist Management
- Cart Management
- Authentication
  - Login
  - Logout
  - Signup
- Single Product Page
- Loading and Alerts
- User Profile Page

<hr />
<br />

## **Tech Stack**

- ReactJS
- CSS
- [Mockbee](https://mockbee.netlify.app/) - Mock Backend
- [Colorant UI](https://colorant-ui.netlify.app) - Component Library
